# reimagined-doodle
Discord bot random team for lol 5v5

A discord bot that randomize people into a even equip.
```
/randomize [voice channel name]
```
Output : 
```
Team 1: 
    - User1
    - User2
    - UserX
    - UserX
    ...
Team 2: 
    - UserX
    - UserX
    - UserX
    - UserX
    ...
```

## Getting started

```
# On macOS and Linux.
curl -LsSf https://astral.sh/uv/install.sh | sh
```

```
uv venv
uv sync
````
